  About Room Booking System
                         This is an online application for reserving the room in library. The students and instructors who are all holding bearcat Id and bearcat email address can use this application to reserve the room. Make use of this application effectively. Be sure to be on time once booked. If you want to change or cancel the booking, you can do it only 3 hours before the booking time.


You Can use Follow Options on this Application
  • Register for the website
  • Book a Study Room
  • Modify the booking before 3 hours
  • Delete booking
  • View your bookings
